Instructions to execute the code:
- Install XAMPP server and connect to the server
- Import the sql code into phpmyadmin.
- Set Servername: localhost, Username: root, Password
- Database name:vehicle_rental so that the connection.php works fine.
- Save the project folder in htdocs of XAMPP folder under Applications and name it as vehiclerental
- Run http://loclahost/vehiclerental in your browser.
- Admin details: Username - root, Password - root
